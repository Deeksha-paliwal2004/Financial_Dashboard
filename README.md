# Financial_Dashboard
 A modern Finance Tracking Dashboard built using React that helps users manage transactions, visualize spending, and gain useful financial insights.


 Features
📊 Dashboard Overview
Total Balance
Total Income
Total Expenses
📈 Insights
Top Spending Category
Monthly Change (%)
Average Transaction Value
Total Transactions Count
🧾 Transaction Management
Add Transaction
Edit Transaction
Delete Transaction
🔍 Filters
Filter by Type (Income/Expense)
Filter by Category
Filter by Date Range
👤 Role-Based Access
Admin → Full access (Add/Edit/Delete)
Viewer → Read-only access
📉 Charts & Visualization
Balance Trend (Line Chart)
Spending Breakdown (Pie Chart)
🛠️ Tech Stack
Frontend: React.js
State Management: Context API
Charts: Recharts
Styling: CSS
